# ✨ About The Project

This is a **Full-Stack MERN Job Portal** that connects **recruiters** and **candidates** in a real-world hiring workflow.

✔ Secure authentication  
✔ Job posting & applications  
✔ Role-based dashboards  
✔ Scalable backend architecture  
✔ Responsive modern UI
